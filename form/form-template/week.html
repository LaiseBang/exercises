This is a demonstration of week - before getting started, replace all references to week with the element you are demonstrating. And also rename the file! And change this text to be a short description of the week

What week would you like to start?

Used for:

Creating input fields allowing to pick weeknumber in a year. In chrome and Opera you get a pop-up calender to select the date. In edge you get week and year pickers with sliding reels.

Important attributes:

size
input type="week" doesn't support form sizing attributes such as size. You'll have to resort to CSS for sizing needs.

Typical child elements:

This element can't have children
